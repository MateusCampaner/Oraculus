Oraculus - Sobre

Desenvolvedor

O sistema Oraculus foi desenvolvido por Mateus Eduardo Campaner, aluno de Engenharia de Software do Centro Universitário Filadélfia - UniFil em Londrina, dentro do NPI - Núcelo de Práticas em Informática.

Sobre o sistema

O sistema Oraculus desenvolvido tem como objetivo principal fornecer uma predição de colheita utilizando o algoritmo de classificação supervisionada KNN, através da inserção de dados de análise de solo contendo dados de Nitrogênio, Fósforo, Potássio, Temperatura, Umidade, pH e Chuva, e retornando a melhor cultura com base nos dados da análise de solo.

Informações técnicas

O sistema Oraculus foi desenvolvido utlizando a linguagem Python ( Versão 3.11.3 ), juntamente com o framework Django ( Versão 4.2 ), foi utilizado a a biblioteca do Bootstrap ( Versão 5.3.2 ) para a estilização do sistema e icones, o sistema também possui uma integração com a bilbioteca do Streamlit ( Versão 1.25.0 ) para a geração de um dashboard com gráficos.
